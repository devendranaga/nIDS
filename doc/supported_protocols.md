# Supported Protocols

The following protocol parsing is currently supported.

| S.No | Protocol Name | Layer |
|------|---------------|-------|
| 1 | Ethernet | L2 |
| 2 | ARP | L2 |
| 3 | VLAN | L2 |
| 4 | IPv4 | L3 |
| 5 | IPv6 | L3 |
| 6 | TCP | L4 |
| 7 | UDP | L4 |
| 8 | ICMP | L4 |
| 9 | ICMP6 | L4 |
| 10 | DHCP | APP |
| 11 | NTP | APP |
