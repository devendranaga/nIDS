# Supported Protocols

The following protocol parsing is currently supported.

| S.No | Protocol Name | Layer |
|------|---------------|-------|
| 1 | Ethernet | L2 |
| 2 | IEEE 802.1AE | L2 |
| 3 | ARP | L2 |
| 4 | VLAN | L2 |
| 5 | IPv4 | L3 |
| 6 | IPv6 | L3 |
| 7 | TCP | L4 |
| 8 | UDP | L4 |
| 9 | ICMP | L4 |
| 10 | ICMP6 | L4 |
| 11 | DHCP | APP |
| 12 | NTP | APP |
| 13 | TLS | APP (only version matching!) |
